# Sitemap 網站架構

## 主要頁面結構

```
首頁
├── 通報/求助
│   ├── 入口
│   │   ├── 通報災情
│   │   ├── 通報/求助
│   │   └── 當志工
│   └── 通報災情 → 地圖
├── 最新消息
│   ├── 媒體報導
│   ├── 政府資訊
│   ├── 最新災情
│   └── 志工徵求
├── 徵求志工
├── 志工資訊
│   ├── 志工團體
│   └── 志工報名填寫入口
├── 徵求配送
├── 常見問題
├── 查詢案件
├── 志工需求
└── 配送需求
```

## 網站結構概念

```
母網站(推廣光復超人) → 維運(光復救災/復原使用)
                              └→ 通用性
```

## 核心模組對應
- 地圖站點 Map → /map
- 志工媒合 Volunteer Tasks → /tasks
- 配送媒合 Delivery → /delivery
- 資訊頁 Info Page → /info
- 審核後台 Moderator Admin → /moderator
- 系統後台 System Admin → /admin
