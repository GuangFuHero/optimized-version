# User Flow 使用者流程

## 主要流程

### 入口分流
```
Start → 首頁 → 身分選擇
                  ├── 團體 → 註冊 → 進入管理介面
                  ├── 個人志工 → 第一次拜訪嗎?
                  │                 ├── Yes → 行前須知 → 地圖
                  │                 └── No → 地圖
                  └── 災民 → 有情報組人嗎?
                              ├── Yes → 情報組人填需求單
                              └── No → 個人填需求單
```

## 使用者角色

### 主要角色
- 災民
- 志工
- 外地志工
- 配送志工
- 資料站點提供者
- 一般用戶
- 志工領隊

## 核心價值
在救災黃金72小時快速啟動網站，協助志工、團體與災區的運作順暢
